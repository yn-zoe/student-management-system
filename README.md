## Project Description
The Student Management System is a simple console-based Python application designed to manage student records.  
It allows users to add new students, view all registered students, and search for a student using their matric number.

---

## Features
- Add a student record
- View all student records
- Search for a student by matric number
- Simple and user-friendly menu interface

- Any operating system (Windows, Linux, macOS)
